\documentclass{article}

这里的documentclass指定了文档的类型，最刚广泛的有article:普通的文章；其他的还有book、report等

如果希望创建一个幻灯片格式的文档则可以使用beamer

需要在文档中显示中文，则需要使用一个叫做ctexart的文档类型，它支持简体中文和英文的混合排，还要指定文档的编码类型，UTF8是TeXworks编辑器默认使用的编码类型，如果不去正确设置它的话，文档中的中文可能会变成一堆乱码

\documentclass[UTF8]{ctexart}

所有位于\begin{document}之前的内容都被称作是前言（preamble），可以在这里设置文档的格式，页面的尺寸，也可以指定文档中需要导入的宏包等等。

在\begin{document} 和\end{document}之间的内容称为文档的正文，在这片区域输入的内容将会被排版到最终生成的文档中。

在前言区块内，可以通过\title{}命令给文档设置一个标题，\author{}命令可以用来指定作者的名字，\date{}命令可以用来指定日期，也可以在此命令中加入\today命令来自动生成当天的日期.

要显示前言部分的信息还需要在文档的正文区域中加入一个\maketitle命令，这个命令会在当前位置生成文档的标题，即前言区(preamble)设定的信息。

Latex中最基础的格式化命令

\textbf{}：加粗文字，需要加粗的文字直接写在花括号中，bf是bold font的缩写代表加粗

\textit{}: 斜体文字，it代表italic(斜体)

\underline{}: 下划线

换行则可以通过两个回车enter,即在正文中空开一行即可。需要注意的是单独的一个换行符只会生成一个空格，并不会开启新的段落。

\section{}: 开启一个新章节

\subsection{}:这是section下的子章节命令，即二级章节

\subsubsection: 三级章节

如果是书籍排版ctexbook, 可以加入比section更大的chapter

\chapter{}: 比secton更大的章节，通常用来表示书籍中的第几章

\part{}: 比chapter还要大的part，通常用来表示书籍中的第几部

如果需要在文章中添加图片，则需要在preamble区域中引用graphicx这个包(注意最后的字符是x)

\usepackage{graphicx}：引用添加图片的包，这个包里包含若干个绘制图片的命令

\includegraphics{}: 在正文中使用此命令在当前位置添加一张图片,花括号中的是图片的名字，在名字中可以直接省略掉后面png的扩展名部分。

若发现添加的图片尺寸过大，超出页面等，可以给\includegraphics{}命令添加一个可选的参数。

\includegraphics[width=0.5\textwidth]{head} 这里的\textwidth代表当前文本区域的宽度

如果想要给图片添加标题，可以将图片嵌套在一个figure环境中,随后我们可以通过\caption{}命令指定图片的标题。同时还能使用\centering命令将图片居中显示。

列表显示

需要切换到列表中的environment(环境)是LaTeX的一个专业术语，即\begin{...}...\end{...}，它像是编程语言中的作用域。位于同一个环境中的内容将会共享相同的文字格式。

itemize:无序列表的创建的环境

\item:列表中每个元素都需以此为开头

enumerate: 对于有序列表(也就是前面带编号的列表)的环境

数学公式

$...$: 行内公式需要写在两个美元符号之间，即文字中插入数学公式无需另起一行

equation: 该环境中可以将公式单独写在一行，且公式行有标号，也可以将环境简写为 \[ ... \]但没有标号。

其他的与markdown中的类似。

表格

tabular：在该环境中创建一个表格

tabular环境要求我们传入一个参数，用来指定表格的尺寸，这里的{ c c c}代表表格一共有三列，c(centering):其中每一列都居中对齐，并且可以替换这里的c为l来表示左对齐, r来表示右对齐。

而下面这个区域中定义了表格中需要显示的内容，其中每列的数据需要以&符号隔开，表格每一行数据需要以\\分割。

也可以在{ c c c} 中添加|变为 { |c|c|c| }，表示这里加入竖线为表格添加竖直的边框，水平方向的边框则需要痛过\hline来表示，甚至可以输入两次\hline即\hline\hline来添加双横线的效果。

如果需要单独指定每一列的宽度可以将c改成p，并在后面的花括号里输入列宽，{ | p{2cm} |c|c| }，这里的p代表paragraph是一种允许设置列宽的列格式。

和图表类似，给表格添加标题，可以先将整个表格放在table环境中,随后可以通过\caption{}命令指定表格的标题，并通过\center命令将表格居中显示。
